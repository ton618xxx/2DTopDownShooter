# 2D Top-Down Shooter 

Template for main project. Unity 2D top-down shooter.

## Setup

### Git LFS (required for binary assets)

This project uses [Git LFS](https://git-lfs.com/) for large binary files (textures, audio, models, etc.). After cloning, run:

```bash
git lfs install
git lfs pull
```

If Git LFS is not installed: download from [git-lfs.github.com](https://git-lfs.github.com/) and run `git lfs install` once.

### Unity

1. Open the project folder in Unity Hub.
2. Use the recommended Unity version (see ProjectSettings if present).
